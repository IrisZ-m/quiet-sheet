# Quiet Sheet

一个 Excel 风格的本地学习工作台，支持日常复习、Anki 文本卡片导入、资料管理和预算合并小游戏。

> 当前版本：v1.0 Beta（2026-07-19）

## 主要功能

- Excel 风格界面、紧凑功能区和全屏显示
- 工作簿名称可在本机自定义
- “今日待办”只显示到期复习与每日新卡，评估后自动移出
- 可按导入批次选择学习范围，并为不同批次分别保存顺序或随机学习方式
- “项目总表”用于搜索、编辑、分类、批量选择和删除
- Anki TXT 与通用 CSV、TSV 导入、表头识别和字段映射
- 12MB 以上大文件提示，最大支持 50MB，并显示读取、解析和记录数量状态
- 自动识别 Anki 的 GUID、笔记类型、牌组、标签等列元数据
- 支持 {{c1::答案}} 完形填空语法；每条源笔记只生成一张学习卡
- 轻量转换段落、列表、表格、Ruby 注音、上下标、代码块和公式源码，自动清理媒体与外部样式
- 每次导入保存文件批次，可筛选或一键删除整个批次
- 可拖动并自动保存表格列宽
- 本地保存学习进度、导入卡片和小游戏最高分
- 自定义 PNG、JPG 或 WebP 工作报表封面
- 完全静态，不需要服务器或数据库

## 本地使用

请下载整个 excel-learning-demo 文件夹，然后双击 index.html。不要只复制 HTML 文件，因为页面还需要同目录中的 CSS 和 JavaScript 文件。

更新后如果仍显示旧界面，请按 Ctrl + F5 强制刷新。

## Anki 导入

在 Anki 中把牌组导出为“笔记（纯文本）”或 UTF-8 CSV、TSV、TXT，然后点击页面顶部的“导入卡片”。

导入器支持：

- 自动识别 Anki 导出文件中的列说明
- 手动调整字段映射
- HTML 转换为安全纯文本
- 完形填空答案提取
- 重复内容更新、跳过或保留副本
- 文件名、导入时间和卡片数量批次记录

旧版本已经导入的卡片没有保存原文件名，会统一显示为“历史导入”。新版本之后的每次导入会单独记录。

当前版本不会导入 Anki 的复习历史，也不支持朗读、图片、音频或视频。媒体引用会被清理；只有媒体而没有有效文字的记录会在预览中标记并跳过。

## 今日待办

今日列表由两部分组成：

- 已到复习日期的卡片
- 当日新卡，默认上限 20 张，可在页面中调整

完成一次评估后，卡片会从当天列表移出，并按照选择安排到 1、2 或 7 天后复习。每日新卡额度会扣除当天已经首次学习的新卡，不会在完成后自动补满。

## 数据与隐私

页面没有账号系统、第三方统计或外部脚本，也不会主动上传学习内容。卡片、学习记录、导入批次、列宽和游戏成绩保存在当前浏览器本机。

清除浏览器网站数据或更换浏览器后，这些记录不会自动迁移。请勿导入公司、客户或其他敏感资料，并遵守所在单位的设备和软件使用规定。

## 部署

这是一个不需要构建步骤的静态网页，可部署到 GitHub Pages、Cloudflare Pages 或其他静态网站托管服务。部署时必须保留 `index.html` 与同目录中的全部 CSS、JavaScript 文件。

本地双击运行时不会联网。部署到托管平台后，浏览器需要联网下载网页文件，但学习数据仍只保存在当前浏览器，不会由本项目代码上传到托管平台。

## 项目文件

- index.html：页面入口
- demo.css、enhancements.css、manager.css：界面样式
- demo.js：基础学习、报表和游戏
- enhancements.js：列宽和 Anki 文本导入
- manager.js：搜索、编辑和批量资料管理
- import-v3.js：Anki、通用 CSV、完形填空和导入诊断
- study-v3.js：项目总表、批次管理、中文搜索和今日复习队列
