# Quiet Sheet

一个 Excel 风格的本地学习工作台，适合在碎片时间背单词、导入 Anki 文本卡片，并包含一个预算合并小游戏。

## 在线体验

启用 GitHub Pages 后，把这里替换成你的在线地址：

`https://你的用户名.github.io/quiet-sheet/`

## 主要功能

- Excel 风格界面与全屏显示
- 商务和日常英语学习
- Anki CSV、TSV、TXT 导入与字段映射
- 可拖动并自动保存的表格列宽
- 本地保存学习进度、导入卡片和游戏最高分
- 自定义工作报表封面
- 完全静态，不需要服务器或数据库

## 本地使用

下载整个项目文件夹，双击 `index.html` 即可。不要只下载单独的 HTML 文件，因为页面还需要同目录中的 CSS 和 JavaScript 文件。

## Anki 导入

在 Anki 中将牌组导出为“笔记（纯文本）”或 UTF-8 CSV/TSV，然后在页面顶部选择“导入卡片”。导入器支持字段映射、内容预览、HTML 清理、填空题和重复卡片处理。

当前版本不导入 Anki 的复习历史；图片和音频仅转换为文字占位提示。

## 数据与隐私

页面没有账号系统，也不会主动上传学习内容。卡片、学习记录、列宽和游戏成绩保存在当前浏览器中。清除浏览器网站数据或更换浏览器后，这些记录不会自动迁移。

在线版本首次打开需要网络，但学习数据仍保存在浏览器本机。请遵守所在单位的设备、网络和软件使用规定，不要导入公司、客户或其他敏感资料。
